# haskell_snake_game

## How to install
### Prerequisites
1. Stack

### Install and Run
> `~ git clone https://github.com/CGQAQ/haskell_snake_game.git`\
> `~ cd haskell_snake_game`\
> `~ stack install`\
> `~ SnakeGame`

 
## Keys
1. Home     (start/pause game)
2. End      (reset game)
3. Up       (move up)
4. Down     (move down)
5. Left     (move left)
6. Right    (move right)
